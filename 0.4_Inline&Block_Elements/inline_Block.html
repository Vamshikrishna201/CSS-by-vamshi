1. Block-Level Elements

Definition:

A block-level element always starts on a new line and typically occupies the entire available horizontal space of its parent container. You can control its height, width, margins, and padding on all sides (top, bottom, left, right).

Common Examples:
div, p, h1 through h6, ul, li, header, footer.

2. Inline Elements

Definition:

An inline element does not start on a new line and only takes up as much width as is necessary for its content. While you can apply horizontal padding and margins, setting a specific height or vertical margin-top/margin-bottom generally has no visual effect.

Common Examples:
span, a, img, strong, em, br.
